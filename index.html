سلسلة النظريات والقواعد في الرياضيات

دليل شامل للقواعد الأساسية مع أمثلة توضيحية

1. المثلثات

مجموع زوايا المثلث = 180°

مجموع الزوايا الداخلية لأي مثلث يساوي دائمًا 180 درجة.

∠A + ∠B + ∠C = 180°

مثال: إذا كانت زوايا مثلث هي 50° و 60°، فإن الزاوية الثالثة = 180 - (50 + 60) = 70°.

نظرية فيثاغورس

في المثلث القائم الزاوية، مربع الوتر يساوي مجموع مربعي الضلعين الآخرين.

c² = a² + b²

مثال: مثلث قائم الزاوية أضلاعه 3 و 4، فإن الوتر = √(3² + 4²) = √(9 + 16) = √25 = 5.

2. الهندسة المستوية (المضلعات)

المربع

المربع له أربعة أضلاع متساوية في الطول وزوايا قائمة.

محيط (P) = 4a

مساحة (A) = a²

المستطيل

المستطيل له أربعة أضلاع متقابلة ومتوازية، وزوايا قائمة.

محيط (P) = 2(l + w)

مساحة (A) = l × w

المعين

المعين له أربعة أضلاع متساوية، والأقطار متعامدة.

محيط (P) = 4a

مساحة (A) = ½ × d₁ × d₂

3. الدائرة

الزاوية المركزية والمحيطية

الزاوية المركزية تساوي ضعف الزاوية المحيطية المشتركة معها في القوس.

∠Central = 2 * ∠Inscribed

محيط ومساحة الدائرة

محيط الدائرة هو المسافة حولها، ومساحتها هي المنطقة التي تشغلها.

محيط (C) = 2πr

مساحة (A) = πr²

4. الهندسة الفراغية

حجم المكعب

المكعب له ستة أوجه مربعة متساوية.

V = a³

حجم الأسطوانة

الأسطوانة لها قاعدتان دائريتان متوازيتان.

V = πr²h

حجم المخروط

المخروط له قاعدة دائرية ورأس واحد.

V = (1/3)πr²h

حجم الكرة

الكرة هي جسم مستدير ثلاثي الأبعاد.

V = (4/3)πr³

5. الجبر

المتطابقات الشهيرة

أهم المتطابقات الجبرية المستخدمة في التحليل والتبسيط.

(a+b)² = a² + 2ab + b²

(a-b)² = a² - 2ab + b²

(a+b)(a-b) = a² - b²

حل المعادلة التربيعية

الصيغة العامة لحل المعادلة ax² + bx + c = 0.

x = [-b ± √(b² - 4ac)] / (2a)

6. التفاضل والتكامل

التفاضل

المشتقة تمثل معدل التغير اللحظي للدالة.

d/dx (xⁿ) = n·xⁿ⁻¹

التكامل

التكامل هو العملية العكسية للتفاضل.

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C

7. الإحصاء والاحتمالات

الوسط الحسابي

متوسط القيم في مجموعة بيانات.

x̄ = Σxᵢ / n

قواعد الاحتمال

القوانين الأساسية لحساب الاحتمالات.

P(A∪B) = P(A) + P(B) - P(A∩B)

P(A∩B) = P(A) × P(B) [إذا كانا مستقلين]
